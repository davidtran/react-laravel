# react-laravel
A simple document reader application. Only PDF file type is allowed to upload and view.

## How to use
```
git clone https://github.com/davidtran/react-laravel
cd react-laravel
composer install
npm run dev
php artisan storage:link
open 127.0.0.1:8000
```
